How to Use These

Three steps. No setup.

Each template is a Typst file. Typst is a free, browser-based typesetting tool — think LaTeX, but actually pleasant to use. You don't install anything. Everything happens in your browser.

  1. Click Download Typst (.typ) on the template you want.
  2. Go to typst.app, sign in (free), and create a new blank project.
  3. Open the downloaded .typ file in any text editor, copy everything, paste into the Typst editor. Edit the placeholder details with your own. The PDF on the right updates as you type — download it when you're done.

The accent colour, font, and section structure are already configured. You're only editing content. If you want to change the colour or move sections around, the file is short and the comments tell you where.

Note: these templates use the basic-resume package by Stuart Foltz, which Typst loads automatically the first time you compile. No manual install required.

A Note on Use

These are formats, not stories.

A clean resume helps. It does not get you in. The bullet grammar in these templates — Category Label, action verb, what you did, bolded metric — is the format I use because it forces clarity. But the metric has to be real, and the work has to matter. The structure is the easy part. The narrative underneath is the hard part, and the part that gets you admitted.

If you want help with that part, that's what I do.
